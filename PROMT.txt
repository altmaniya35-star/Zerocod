
Напиши Python-скрипт, который:

Читает все доступные CSV и JSON файлы в директории /data.
CSV - парсит с пюмощью pandas или Csv.
JSON - парсит стандартной библиотекой json.
Читает все HTML-шаблоны из директории /templates.
При запуске выводит список:
доступных файлов с данными;
доступных шаблонов.
Дает пользователю выбрать (через input) один файл данных и один HTML-шаблон.
Выводит список доступных чеков (по invoice id) из выбранного файла.
Пользователь выбирает конкретный invoice id, и по нему генерируется PDF-документ:
данные подставляются в HTML-шаблон;
PDF сохраняется в директории /output.
После генерации автоматически открывает PDF в системной программе.
Требования:

Используй библиотеку WeasyPrint для генерации PDF.
Поддержка кириллицы (подключи шрифт DejaVu Sans или Roboto).
Скрипт должен работать на Windows и macOS.
Сделай аккуратный вывод меню в консоли (нумерация вариантов).
Создай сопроводительные файлы Readme, requirments. Добавь комментарии на русском языке и объясни код. 
Сгенерируй файлы CSV, JSON, HTML для customer, product, invoice. 
В invoice в заголовке должна отображаться информация по счёту - номер, дата, а также информация о покупателе. В строках счёта информация о товаре, цене за штуку, сумме. Итог по сумме расположить после всех позиций в счёте. Количество строк на одной странице счета 10 штук, если строк больше тогда переносить на следующую страницу где также должна быть шапка счёта. Пронумеровать страницы. 
Товары будут штучные (фирма торгует комплектующими для персональных компьютеров - учесть это при наполнении таблицы с товарами). Также наполнить файл CSV и JSON по customer любой одной записью.
